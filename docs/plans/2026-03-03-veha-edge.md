# veha-edge Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a single pre-built `veha-edge` binary that self-installs on an edge device, supervises the player as a subprocess, and runs the agent — replacing the compile-on-device install flow with a 30-second setup.

**Architecture:** New `veha-edge` Rust crate added to the workspace. `veha-agent` and `veha-player` are refactored into library + binary pairs so `veha-edge` can import their logic. The `run` subcommand spawns `veha-edge player` as a child process (preserving `!Send` OutputSink isolation) while running agent logic in the same tokio runtime. GitHub Actions CI cross-compiles for x86_64 and aarch64 and uploads binaries to GitHub Releases.

**Tech Stack:** Rust 1.85+ edition 2024, clap 4 (derive, subcommands), tokio (full), reqwest 0.12 (update download), cross (aarch64 cross-compilation in CI), GitHub Actions.

**Important notes:**
- `ws_client::run` in veha-agent already has the right signature: `pub async fn run(config: AgentConfig)`. It never returns — it's an infinite reconnect loop. This is intentional.
- The player's `async fn main()` at line 160 of `veha-player/src/main.rs` contains all the logic. We extract it into `pub async fn run(config: PlayerConfig)` in lib.rs.
- `veha-player` has a `framebuffer` feature gate for the framebuffer output backend. `veha-edge` must forward this feature: `framebuffer = ["veha-player/framebuffer"]`.
- Config field names differ between agent and player — `player_socket` (agent) vs `socket_path` (player). The merged `EdgeConfig` uses `player_socket` and we map it when constructing `PlayerConfig`.

```rust
use clap::{Parser, Subcommand};

#[derive(Parser)]
#[command(
    name = "veha-edge",
    about = "koompi-veha edge device — agent + player in one binary",
    version
)]
struct Cli {
    #[command(subcommand)]
    command: Command,
}

#[derive(Subcommand)]
enum Command {
    /// First-time device setup: install binary, config, and systemd service.
    Install,
    /// Remove the systemd service, config, and binary.
    Uninstall {
        /// Also remove the media cache at /var/cache/veha.
        #[arg(long)]
        purge: bool,
    },
    /// Main runtime entry point (used by systemd). Runs agent + supervises player.
    Run,
    /// Player-only entry point (spawned by `run` as a subprocess).
    Player {
        #[arg(short, long, default_value = "/etc/veha/veha-edge.toml")]
        config: String,
    },
    /// Download and install the latest veha-edge release from GitHub.
    Update,
    /// Show current service and player status.
    Status,
}

#[tokio::main]
async fn main() {
    tracing_subscriber::fmt::init();

    let cli = Cli::parse();

    match cli.command {
        Command::Install => install::run(),
        Command::Uninstall { purge } => install::uninstall(purge),
        Command::Run => run::run().await,
        Command::Player { config } => run::player(&config).await,
        Command::Update => update::run(),
        Command::Status => status::run(),
    }
}
```

```rust
use std::fs;
use std::process::Command;

use tracing::{error, info};

struct InstallParams {
    board_id: String,
    board_name: String,
    api_url: String,
    api_key: String,
    output_backend: String,
    width: u32,
    height: u32,
}

fn read_install_params() -> Result<InstallParams, String> {
    let board_id = std::env::var("BOARD_ID")
        .map_err(|_| "BOARD_ID env var is required".to_string())?;
    let server_url = std::env::var("SERVER_URL")
        .map_err(|_| "SERVER_URL env var is required".to_string())?;

    // Derive WebSocket URL from HTTP server URL
    let api_url = server_url
        .replace("https://", "wss://")
        .replace("http://", "ws://")
        + "/ws/agent";

    Ok(InstallParams {
        board_name: std::env::var("BOARD_NAME").unwrap_or_else(|_| board_id.clone()),
        api_key: std::env::var("API_KEY").unwrap_or_default(),
        output_backend: std::env::var("OUTPUT_BACKEND")
            .unwrap_or_else(|_| "framebuffer".into()),
        width: std::env::var("WIDTH")
            .ok()
            .and_then(|v| v.parse().ok())
            .unwrap_or(1920),
        height: std::env::var("HEIGHT")
            .ok()
            .and_then(|v| v.parse().ok())
            .unwrap_or(1080),
        board_id,
        api_url,
    })
}

fn render_config_toml(p: &InstallParams) -> String {
    format!(
        r#"# Generated by veha-edge install — edit as needed

board_id = "{board_id}"
board_name = "{board_name}"
api_url = "{api_url}"
api_key = "{api_key}"
output_backend = "{output_backend}"
width = {width}
height = {height}
fullscreen = true
player_socket = "/run/veha/player.sock"
cache_dir = "/var/cache/veha"
report_interval_secs = 10
screenshot_interval_secs = 60
player_restart_delay_secs = 5
"#,
        board_id = p.board_id,
        board_name = p.board_name,
        api_url = p.api_url,
        api_key = p.api_key,
        output_backend = p.output_backend,
        width = p.width,
        height = p.height,
    )
}

fn render_systemd_unit() -> String {
    r#"[Unit]
Description=veha-edge board agent + player
After=network-online.target
Wants=network-online.target

[Service]
ExecStart=/opt/veha/veha-edge run
Restart=on-failure
RestartSec=5
Environment=RUST_LOG=info

[Install]
WantedBy=multi-user.target
"#
    .into()
}

fn render_tmpfiles_conf() -> String {
    "d /run/veha 0755 root root -\n".into()
}

pub fn run() {
    let params = match read_install_params() {
        Ok(p) => p,
        Err(e) => {
            error!("{e}");
            std::process::exit(1);
        }
    };

    info!("Installing veha-edge for board_id={}", params.board_id);

    // 1. Copy binary to /opt/veha/
    let exe = std::env::current_exe().expect("cannot resolve current exe path");
    fs::create_dir_all("/opt/veha").expect("failed to create /opt/veha");
    fs::copy(&exe, "/opt/veha/veha-edge").expect("failed to copy binary to /opt/veha/veha-edge");
    info!("Binary installed to /opt/veha/veha-edge");

    // 2. Write config
    fs::create_dir_all("/etc/veha").expect("failed to create /etc/veha");
    fs::write("/etc/veha/veha-edge.toml", render_config_toml(&params))
        .expect("failed to write /etc/veha/veha-edge.toml");
    info!("Config written to /etc/veha/veha-edge.toml");

    // 3. Create cache dir
    fs::create_dir_all("/var/cache/veha").expect("failed to create /var/cache/veha");

    // 4. Write tmpfiles.d entry (creates /run/veha on boot)
    fs::write("/etc/tmpfiles.d/veha.conf", render_tmpfiles_conf())
        .expect("failed to write tmpfiles.d config");
    // Also create the run dir now so we don't need a reboot
    fs::create_dir_all("/run/veha").ok();

    // 5. Write systemd unit
    fs::write(
        "/etc/systemd/system/veha-edge.service",
        render_systemd_unit(),
    )
    .expect("failed to write systemd unit");
    info!("Systemd unit written");

    // 6. Enable and start
    for args in [
        vec!["daemon-reload"],
        vec!["enable", "--now", "veha-edge"],
    ] {
        let status = Command::new("systemctl")
            .args(&args)
            .status()
            .expect("failed to run systemctl");
        if !status.success() {
            error!("systemctl {} failed", args.join(" "));
            std::process::exit(1);
        }
    }

    println!("\nveha-edge installed and started successfully.");
    println!("  Board ID : {}", params.board_id);
    println!("  API URL  : {}", params.api_url);
    println!("\nCheck logs: journalctl -u veha-edge -f");
}

pub fn uninstall(purge: bool) {
    info!("Uninstalling veha-edge");

    for args in [
        vec!["disable", "--now", "veha-edge"],
    ] {
        Command::new("systemctl")
            .args(&args)
            .status()
            .ok();
    }

    let paths = [
        "/etc/systemd/system/veha-edge.service",
        "/etc/tmpfiles.d/veha.conf",
        "/etc/veha/veha-edge.toml",
        "/opt/veha/veha-edge",
    ];
    for path in &paths {
        fs::remove_file(path).ok();
    }
    fs::remove_dir("/etc/veha").ok();
    fs::remove_dir("/opt/veha").ok();

    if purge {
        fs::remove_dir_all("/var/cache/veha").ok();
        info!("Media cache removed");
    }

    Command::new("systemctl").arg("daemon-reload").status().ok();

    println!("veha-edge uninstalled.");
    if !purge {
        println!("Media cache kept at /var/cache/veha. Use --purge to remove it.");
    }
}
```

```rust
use std::path::{Path, PathBuf};
use std::time::Duration;

use tracing::{error, info, warn};

use crate::config::EdgeConfig;

const CONFIG_PATH: &str = "/etc/veha/veha-edge.toml";

/// Main runtime entry point. Runs agent + supervises player subprocess.
pub async fn run() {
    let config = load_config(CONFIG_PATH);
    let agent_config = veha_agent::AgentConfig::from(&config);
    let exe = std::env::current_exe().expect("cannot resolve current exe");
    let restart_delay = Duration::from_secs(config.player_restart_delay_secs);

    tokio::join!(
        supervise_player(exe, restart_delay),
        veha_agent::run(agent_config),
    );
}

/// Player-only entry point (spawned by `run` as a subprocess).
pub async fn player(config_path: &str) {
    let config = load_config(config_path);
    let player_config = veha_player::PlayerConfig::from(&config);

    veha_core::init().expect("FFmpeg initialization failed — check FFmpeg libraries are installed");
    veha_player::run(player_config).await;
}

fn load_config(path: &str) -> EdgeConfig {
    if Path::new(path).exists() {
        EdgeConfig::from_file(path).unwrap_or_else(|e| {
            error!("Failed to load config from {path}: {e}");
            std::process::exit(1);
        })
    } else {
        error!("Config file not found: {path}");
        std::process::exit(1);
    }
}

async fn supervise_player(exe: PathBuf, restart_delay: Duration) {
    loop {
        info!("Starting player subprocess");

        let result = tokio::process::Command::new(&exe)
            .arg("player")
            .status()
            .await;

        match result {
            Ok(status) if status.success() => {
                info!("Player exited cleanly — restarting in {}s", restart_delay.as_secs());
            }
            Ok(status) => {
                warn!(
                    "Player exited with status {} — restarting in {}s",
                    status,
                    restart_delay.as_secs()
                );
            }
            Err(e) => {
                error!("Failed to spawn player: {e} — retrying in {}s", restart_delay.as_secs());
            }
        }

        tokio::time::sleep(restart_delay).await;
    }
}
```

**Step 1: Implement `veha-edge/src/update.rs`**

```rust
use std::fs;
use std::os::unix::fs::PermissionsExt;
use std::process::Command;

use tracing::{error, info};

/// GitHub repository (owner/repo) for release downloads.
const REPO: &str = "rithythul/koompi-veha";

/// Installed binary path.
const INSTALL_PATH: &str = "/opt/veha/veha-edge";

fn arch_suffix() -> &'static str {
    if cfg!(target_arch = "x86_64") {
        "x86_64-linux"
    } else if cfg!(target_arch = "aarch64") {
        "aarch64-linux"
    } else {
        panic!("unsupported architecture for self-update");
    }
}

pub fn run() {
    let suffix = arch_suffix();
    let url = format!(
        "https://github.com/{REPO}/releases/latest/download/veha-edge-{suffix}"
    );

    info!("Downloading latest veha-edge from {url}");
    println!("Downloading {url} ...");

    let response = reqwest::blocking::get(&url).unwrap_or_else(|e| {
        error!("Download failed: {e}");
        std::process::exit(1);
    });

    if !response.status().is_success() {
        error!("Download failed with HTTP {}", response.status());
        std::process::exit(1);
    }

    let bytes = response.bytes().unwrap_or_else(|e| {
        error!("Failed to read response body: {e}");
        std::process::exit(1);
    });

    // Write to a temp file first, then atomically replace
    let tmp_path = format!("{INSTALL_PATH}.tmp");
    fs::write(&tmp_path, &bytes).unwrap_or_else(|e| {
        error!("Failed to write to {tmp_path}: {e}");
        std::process::exit(1);
    });

    // Make executable
    fs::set_permissions(&tmp_path, fs::Permissions::from_mode(0o755))
        .expect("failed to set permissions");

    // Atomic replace
    fs::rename(&tmp_path, INSTALL_PATH).unwrap_or_else(|e| {
        error!("Failed to replace binary: {e}");
        std::process::exit(1);
    });

    info!("Binary updated at {INSTALL_PATH}");

    // Restart service
    Command::new("systemctl")
        .args(["restart", "veha-edge"])
        .status()
        .ok();

    println!("veha-edge updated and service restarted.");
}
```
